# nodejs-auth-example
A sample nodejs application with local authentication. 

# Deploy with Docker
1. Run ```docker-compose build && docker-compose up```
2. Open ```http://localhost:3000```

Shoutout to Scotch.io for putting together an awesome node-js auth tutorial -
